# Lista de Dispositivos – Condomínio de Laboratórios CCAA

Este diretório apresenta a **lista consolidada de dispositivos** que compõem a infraestrutura do Condomínio de Laboratórios do Centro de Ciências Agrárias Aplicadas (CCAA), considerando o uso multiusuário, a padronização dos ambientes e os limites financeiros estabelecidos pelo Edital.

A lista foi elaborada para subsidiar o **dimensionamento técnico**, a **orçamentação** e o processo de aquisição por meio de Atas de Registro de Preços.

---

## 1. Dispositivos de Computação
- Computadores de bancada (uso discente e docente)
- Estações de trabalho para análise de dados

## 2. Infraestrutura de Rede
- Switches de acesso (camada 2)
- Cabeamento estruturado (Cat 6)
- Patch cords e pontos de rede
- Infraestrutura de acesso à rede Wi-Fi corporativa

## 3. Infraestrutura de Servidores (Uso Compartilhado)
- Servidor de aplicação
- Servidor de banco de dados
- Sistema de armazenamento centralizado

## 4. Dispositivos IoT e Monitoramento
- Sensores de umidade do solo
- Sensores de pH do solo
- Sensores de temperatura ambiente
- Sensores de umidade relativa do ar
- Sensores de vazão
- Sensores de nível de água
- Gateway IoT

## 5. Infraestrutura de Energia e Suporte
- Nobreaks (UPS)
- Rack de telecomunicações
- Sistema de refrigeração do ambiente técnico

---

## Observações
- Os quantitativos detalhados por laboratório encontram-se nos **repositórios individuais** de cada unidade participante.
- A consolidação orçamentária dos dispositivos está documentada na pasta **`orcamento/`** do repositório do condomínio.
- Todos os dispositivos listados destinam-se exclusivamente à **infraestrutura de pesquisa**, conforme permitido pelo Edital.
