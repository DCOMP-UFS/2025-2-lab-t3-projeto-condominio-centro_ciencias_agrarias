# Plano de Uso Multiusuário do Condomínio

O Condomínio de Laboratórios do CCAA opera em regime de uso compartilhado entre os laboratórios participantes, mediante regras comuns de acesso e utilização.

O uso da infraestrutura será organizado por meio de agendamento prévio, respeitando a prioridade de atividades acadêmicas, projetos de pesquisa em andamento e ações de ensino.

Cada laboratório é responsável pelo uso adequado dos equipamentos durante suas atividades, cabendo à coordenação do condomínio a manutenção preventiva, controle patrimonial e acompanhamento dos indicadores de utilização.

O acesso é permitido a docentes, técnicos e discentes vinculados aos grupos de pesquisa participantes, conforme normas institucionais vigentes.
